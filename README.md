### Quick Start
```bash
# cd backend
cd backend
# start Postgres DB
docker-compose up --build
# run Spring Boot Application
mvn spring-boot:run

# cd frontend
cd frontend
# install dependencies
npm install
# run Angular Application
npm start

# open in browser
http://localhost:4200/
```
